# Codebie

An online judge to practise problems

## Built With

* [VS Code](https://code.visualstudio.com)

## Languages Used

* **Front-End**
  * HTML
  * CSS
  * Bootstrap

* **Back-End**
  * Javascript
  * Node.js
  * Express.js
  * MongoDB

## API Used

* [Judge0](https://rapidapi.com/hermanzdosilovic/api/judge0)

## Supported Compilers

 * C (GCC 9.2.0)
 * C++ (GCC 7.4.0)
 * Java (OpenJDK 13.0.1)
 * Python (3.8.1)
